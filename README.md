# BGTS - Business & Global Technology Solutions

A modern, high-performance corporate website for BGTS, a leading technology solutions provider. This project showcases a premium digital presence using the latest web technologies.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Start-Up-Yazilim/bgts-web
    cd bgts-web
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```
